# Práctica Final - Explotación Semántica de Datos

**Autora:** Laura Llamas López  
**Institución:** Universidad de Murcia  
**Año:** 2026

## Descripción

Grafo de conocimiento RDF sobre el papel de los genes del reloj circadiano humano en los trastornos del ritmo circadiano del sueño. El proyecto modela 11 genes (CLOCK, BMAL1, PER1/2/3, CRY1/2, NR1D1, CSNK1D/E, NPAS2), 7 variantes patogénicas y 5 trastornos del sueño, utilizando ontologías biomédicas estándar (BioLink Model, Relations Ontology...).

## Contenido del repositorio

- **grafo_rdf**: Grafo RDF principal en formato Turtle 
- **consultas_sparql**: Consultas SPARQL para análisis del grafo 
- **documentación**: Capturas de pantalla del despliegue en Blazegraph, resultado de validación W3C...
- **informe**
